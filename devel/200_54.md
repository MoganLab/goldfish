# [200_54] gf doc 遇到已存在但无文档的库或函数时，给出专门提示

## 如何测试

### 1. 函数已被 export，但没有对应函数文档和测试用例
```
./bin/gf doc "path?"
```
预期会输出
```
Function path? is exported in:
  (liii path)
No documentation and test cases available.
Try one of these commands:
  gf doc liii/path
  gf source liii/path
```

### 2. 指定库查询函数时，函数已被 export，但没有对应函数文档和测试用例
```
./bin/gf doc liii/path "path?"
```
预期会输出
```
Function path? is exported in:
  (liii path)
No documentation and test cases available.
Try one of these commands:
  gf doc liii/path
  gf source liii/path
```

### 3. 库存在，但没有对应库级测试文档
```
./bin/gf doc scheme/base
```
预期会输出
```
Library (scheme base) exists.
No documentation and test cases available.
Try one of these commands:
  gf source scheme/base
```

### 4. 库本身不存在时，仍然保持原有提示
```
./bin/gf doc scheme/bash
```
预期会输出
```
Error: library not found in *load-path*: scheme/bash
```

### 5. 相关测试
```
./bin/gf test tests/liii/golddoc
```
