## 5. 2026-05-24 实现 vector-binary-search
### 5.1 What
在 `goldfish/srfi/srfi-133.scm` 中实现了 `vector-binary-search` 函数。

1. 在 `srfi-133.scm` 中导入 `(scheme case-lambda)` 以支持多分支函数定义
2. 实现了支持 3/4/5 个参数的重载版本：
   - `(vector-binary-search vec value cmp)`
   - `(vector-binary-search vec value cmp start)`
   - `(vector-binary-search vec value cmp start end)`
3. 测试通过 `bin/gf test --changed-since=main` 验证

### 5.2 Why
`vector-binary-search` 是 SRFI 133 规范中定义的搜索函数之一，goldfish 此前尚未实现。该函数对已排序向量进行二分查找，是向量库的重要组成部分。

### 5.3 How
参考 SRFI 133 参考实现和 goldfish 已有的 `flexvector-binary-search` 实现，使用 `case-lambda` 处理可选的 `start` 和 `end` 参数，在指定范围内执行标准二分查找算法。比较函数 `cmp` 接受 `(value element)` 两个参数，返回负数/零/正数表示大小关系。
