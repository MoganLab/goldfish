## 5. 2026/05/24 实现 vector-concatenate

### 5.1 What
在 Goldfish 的 SRFI 133 库中实现 `vector-concatenate` 函数。

1. 在 `goldfish/srfi/srfi-133.scm` 中实现 `vector-concatenate`
2. 在 `goldfish/liii/vector.scm` 中导出
3. 添加 `tests/liii/vector/vector-concatenate-test.scm` 单元测试

### 5.2 Why
`vector-concatenate` 是 SRFI 133 标准库中的函数，用于将列表中的多个向量连接成一个新向量。补充该函数以完善 SRFI 133 向量库的支持。

### 5.3 How
通过 `apply vector-append ls` 直接实现，复用已有的 `vector-append` 函数，并补充参数校验：

```scheme
(define (vector-concatenate ls)
  (unless (list? ls)
    (error 'type-error "vector-concatenate: argument is not a list")
  )
  (for-each (lambda (v)
              (unless (vector? v)
                (error 'type-error "vector-concatenate: list element is not a vector")
              ))
            ls)
  (apply vector-append ls)
) ;define
```

该实现利用 Scheme 内置的 `vector-append` 和 `apply` 完成向量的批量连接，同时通过 `list?` 和 `vector?` 对参数进行类型检查，确保传入的是向量列表。
