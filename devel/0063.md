## 5. 2026-05-24 实现 vector-unfold! 和 vector-unfold-right!
### 5.1 What
在 `goldfish/srfi/srfi-133.scm` 中实现了 `vector-unfold!` 和 `vector-unfold-right!` 函数。

1. `vector-unfold!` - 向已有向量指定区间从左到右填充
   - `(vector-unfold! f vec start end initial-seed ...)`
   - 对偏移索引 k（0 到 end-start-1），应用 `f` 到 k 和当前 seeds
   - `f` 返回 n+1 个值：元素和新的 seeds
   - 元素依次写入 vec 的 `[start, end)` 区间

2. `vector-unfold-right!` - 向已有向量指定区间从右到左填充
   - `(vector-unfold-right! f vec start end initial-seed ...)`
   - 对偏移索引 k（0 到 end-start-1），应用 `f` 到 k 和当前 seeds
   - `f` 返回 n+1 个值：元素和新的 seeds
   - 元素逆序写入 vec 的 `[start, end)` 区间（第一个元素放在 end-1，最后一个放在 start）

3. 在 `goldfish/liii/vector.scm` 中导出这两个函数
4. 添加测试文件 `tests/liii/vector/vector-unfold-bang-test.scm` 和 `tests/liii/vector/vector-unfold-right-bang-test.scm`
5. 全部 56 个 vector 测试文件通过验证

### 5.2 Why
`vector-unfold!` 和 `vector-unfold-right!` 是 SRFI 133 规范中定义的 mutator 函数，goldfish 此前尚未实现。这两个函数是 `vector-unfold` / `vector-unfold-right` 的破坏性版本，允许高效地向已有向量的指定区间填充元素，避免额外分配新向量。

### 5.3 How
参考 SRFI 133 规范和 goldfish 已有的 `vector-unfold` / `vector-unfold-right` 实现：
- 使用 `let-values` 处理 `f` 返回的多值结果
- `vector-unfold!` 使用偏移索引 k（从 0 开始），写入位置为 `start + k`
- `vector-unfold-right!` 同样使用偏移索引 k，写入位置为 `end - 1 - k`
- 循环在生成 `(- end start)` 个元素后终止
- 返回值为 `unspecified`（用 `(if #f #f)` 表示）
