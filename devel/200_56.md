# [200_56] check 新增 check-approx 浮点近似断言语法

## 如何测试

### 1. 使用 `check-approx` 断言浮点近似结果
```scheme

```
预期该断言会被纳入 `check-report` 统计，并在误差满足容差时通过。

## 说明

本次新增的语法为：

```scheme
(check-approx expr => expected :rel-tol rel-tol :abs-tol abs-tol)
```

其中：

- `:rel-tol` 是相对误差容差，默认值为 `1e-12`
- `:abs-tol` 是绝对误差容差，默认值为 `1e-12`

判断规则为：

```scheme
abs(actual - expected) <= max(abs-tol, rel-tol * max(abs(actual), abs(expected)))
```

`check-float` 仍然保留，用于兼容旧代码；但新的测试建议优先使用 `check-approx`，因为它会直接接入现有 `check-report` 统计链路。
