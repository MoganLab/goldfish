## 2026/04/22 将 tbox 升级到 1.8.0

### What
将 tbox 从 1.7.7 升级到 1.8.0，并同步更新 GitHub Actions 中的 xmake 版本。

1. 更新 `xmake.lua` 中的 `TBOX_VERSION` 从 `1.7.7` 到 `1.8.0`
2. 更新 `xmake/packages/t/tbox/xmake.lua` 中的版本声明
3. 更新所有 GitHub Actions workflow 中的 xmake 版本从 `v2.9.9` 到 `v3.0.8`
4. 更新 `package.yml` 中 deb 模板的基础 URL 到 `v3.0.8`

### Why
tbox 1.8.0 需要 xmake v3.0.5+ 才能构建，而当前 CI 使用的是 v2.9.9，会导致构建失败。

### How
- tbox 1.8.0 的 `xmake.lua` 中设置了 `set_xmakever("3.0.5")`
- 本地使用 scoop 安装 xmake v3.0.8 验证构建通过
- 同步更新所有 CI workflow 确保云端构建环境一致
